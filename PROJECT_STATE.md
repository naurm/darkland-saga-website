# Project State — Darkland Saga Author Website

## Project Goal
Build an author website for J.L. Allred that presents the author first, then the Darkland Saga world. Multi-page Next.js site with a warm, weathered aesthetic (emberlight tones, storied textures).

## Site Structure
- `/` — Author homepage (done ✅)
- `/books` — Series pages: Darkness Kindled + Hunting Misfortune
- `/world-of-eadrom` — Deep immersive lore experience (adapted from existing 7-chamber prototype)
- `/about` — Author bio, writing philosophy

## Design Foundation
- Next.js 14 + TypeScript + Tailwind CSS 3
- Custom color palette: ember (amber tones), parchment (warm neutrals), emberglow (highlight)
- Fonts: Inter (body), DM Serif Display (headings), JetBrains Mono (data)
- Dark theme, warm tones, textured backgrounds

## Completed
- Next.js project scaffolded and building successfully
- Custom Tailwind design system with ember/parchment/emberglow tokens
- Root layout with fixed nav header (Books, The World, About)
- Author homepage with hero, featured books grid, tagline quote
- Project cognition files (`PROJECT_STATE.md`, `data/worldbuilding-reference.txt`)
